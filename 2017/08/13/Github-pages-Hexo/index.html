Hexo 配置方面

  • 具体可参照官方文档

  • 安装淘宝源镜像的npm来替代官方源npm

    $ npm install -g cnpm --registry=https://registry.npm.taobao.org

    回车

    安装hexo软件包

    $ cnpm install hexo

    接着本地安装Hexo

    输入

    $ npm install -g hexo

_config.yml下部署deploy

deploy:

- type: git

- repo: https://github.com/你的id/你的id.github.io.git

- branch: master

补充

执行$ hexo d出现Error deployer not found:git的处理方法

$ npm install hexo-deployer-git --save
